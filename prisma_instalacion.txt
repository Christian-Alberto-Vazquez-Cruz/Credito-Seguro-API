INSTALACIÓN DE PRISMA

1.- npm init -y
2.- npm install typescript tsx @types/node --save-dev
3.- npx tsc --init
4.- npm install -D prisma@6.19.0
5.- npx prisma init --datasource-provider sqlserver --output ../generated/prisma (o npx prisma init)
6-  npx prisma generate

7.- npm install @prisma/client@6.19.0 && npm install @prisma/adapter-mssql && npm install dotenv


-----------------
Una vez realizados los pasos anteriores colocar la cadena de conexión en .env
